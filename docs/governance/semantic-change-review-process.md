# Semantic Change Contribution and Review Process

Applies to changes affecting:

- Spec DSL semantics
- Core IR semantics/typing
- migration semantics
- conformance requirements

## Required PR Contents

1. Semantic delta summary.
2. Updated normative docs.
3. Negative tests for rejected behavior.
4. Conformance tests for accepted behavior.

## Review Gates

- At least one semantic maintainer approval.
- No unresolved invariance objections.
- Roadmap checkbox linkage for traceability.
